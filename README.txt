#------- Toolの説明について ------#

1, PlotInnerRing.py
2mm球のフィッティングをするためのプログラム。
基本的には、合わなかったデータについて解析するために使う。
フィッティングに使うデータ範囲を調整してうまくやること。

2, PlotOuterRing.py
ビームの中心を求めるためのプログラム。
基本的には合わなかったデータについて解析するために使う。
フィッティングに使うデータ範囲を調整してうまくやること。

3, PlotFilmImage.py
2mm球とビームの中心を同時にフィッティングして、絵を出力するプログラム。
各データの絵が欲しい時に使う。

4, AnalyzeWinstonLutz.py
全てのデータについて2mm球とビームの中心を求めて、Summaryを出力するプログラム。
ついでに絵を描くときは、-pで指定してあげる。
